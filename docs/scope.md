# Project Name: Turnback Tape Audio Landing Page



## Project Purpose:
The purpose of this project to provide a community resource to people who are interested in reviving analog sound, most specifically lo-fi music production. Turnback Tape Audio will operate as a fictional business that will offer customers the ability to take their current mp3 files and convert them into high-bias cassette tapes. The site will serve as a "coming soon" business launch page, where future patrons can sign up with their email and be notified when both the site and business goes live.

## Description:
As the acting unofficial project manager for this site I will develop a one-page website prototype for the fictional business Turnback Tape Audio. The functionality will be limited to just an email sign-up form. The `index.html` file will contain all HTML elements, with a brief description the company's mission and dedication to reviving the warm analog, lo-fi sound associated with cassette tapes. The `styles.css` file will contain a theme and color scheme(`#D4B86A`, `#000000`,`#808080`,`#D22B2B`)modeled after the Maxwell XLII cassette tape, with the goal of showcasing and inspiring 90's nostalgia. 

## Desired Results:
A functional two page website housing an email sign up form for Turnback Tape Audio. This site will also have an href element linking this project to my GitHub account. 

**Root Directory Files:**

*  `README.md`: Professional homepage for the repository linking to the live site and docs
*  `index.html`: Main landing page with the email sign-up form
*  `about.html`: Second page describing Turnback Tape Audio as a business
*  `style.css`: Single external stylesheet containing all of my Maxwell-themed design rules
    
**Documentation Folder (`docs/`):**

**Below is bit more details on the objectives I have for the styling:**

*  Black: `#000000`- will be used for text and logs
*  Grey: `#808080`- will be used for any responsive containers
*  Red: `#D22B2B`- will be used for another accent color and on the "call to action" button


## Exclusions:

This page will be composed exclusively of HTML and CSS. Although the "call to action" button should warrant a response, I will not be using any JavaScript on the initial state of this site. A secondary goal is to return and get it fully operational with JavaScript and mailing list tools.
