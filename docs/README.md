# Turnback Tape Audio: Pre-Launch Prototype

## Project Overview
Turnback Tape Audio is a fictional service designed for lo-fi music producers and analog enthusiasts. This website serves as a "Coming Soon" landing page for a professional MP3-to-cassette conversion service. 

This project demonstrates the culmination of project management principles from *Project Management for the Unofficial Project Manager* and the importance of having the "hybrid generalist" mindset from *Range*.

## Technical Details
* **Architecture**: HTML5 and vanilla CSS3.
* **Branding**: Analog theme inspired by 1990s Maxwell XLII high-bias cassette aesthetics.
* **Typography**: Orbitron (Headings) and Montserrat (Body) via Google Fonts API.
