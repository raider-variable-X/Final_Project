# Project Retrospective: Turnback Tape Audio

## 1. What went well?
* **Visual Identity**: Being able to integrate the Maxwell XLII color palette and retro typography successfully captured the 90s high-fidelity analog audio theme.
* **Project Management**: Following the Work Breakdown Structure (WBS) allowed for me to have a steady development pace. This allowed me to ensure that the documentation and site build were all completed before the March 1st deadline.
* **Technical Implementation**: I was able to successfully utilize the Google Fonts API and display custom CSS skills to create a pre-launch landing page without involving the use of CSS frameworks, JavaScript, and remaining in the project's technical scope.

## 2. What didn't go as planned?
* **CSS Complexity**: I wanted to try to include a "Arc Raiders"(video game) style accent to the sign up element. This became problematic not only because it was outside of the original scope, but it required more time to implement than I initially anticipated.
* **Font Rendering**: It has been a while since I've used Google's Font API so when I noticed that the <link> for web initially only displayed the <uniquifier> that went into the CSS file instead of my <head> section, I was confused. However, when the page was refreshed, the embedded HTML code that you typically see in <head> sections of index.html files finally showed.

## 3. What would you do differently next time?
* **Additional Time**: I am disappointed about not being able to integrate the many design element ideas that I had after my scope was approved. If there was a way to have had more time and foresight in how long it would have taken, then I would have found a way to include those in this original build.
* **CSS Framework Utilization**: Although I am happy with the overall appearance of the site, adding a CSS Framework would make it appear more polished and even more 90's analog sound adjacent.
